Herramientas y Software

Estos son algunos de los programas de software libre y herramientas online que utilizamos en nuestras clases para desarrollar los proyectos.

FreeCAD

Software de diseño asistido por ordenador (CAD) para modelado 3D de piezas y ensamblajes.

Sitio Web

Arduino IDE

Entorno de desarrollo para programar nuestras placas controladoras en proyectos de robótica y electrónica.

Sitio Web

Tinkercad

Plataforma online para el diseño 3D intuitivo y la simulación de circuitos electrónicos con Arduino.

Sitio Web

GIMP

Editor de imágenes y gráficos rasterizados. Una potente alternativa de software libre para la edición fotográfica.
